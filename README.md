## How to use?
### prerequisites
* NodeJs ![How to install?](https://nodejs.org/en/download)
* Git ![How to install?](https://git-scm.com/downloads)
### How to run
* Clone the repository `git clone `
* Change directory to the repo `cd `
* Run the script `node app.js`
